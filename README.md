##  Технологический стек

### Backend
- **Язык**: Go 1.26
- **Фреймворк**: Gin
- **ORM**: GORM + PostgreSQL
- **Аутентификация**: JWT 
- **Кеширование**: Redis
- **Хеширование**: bcrypt

### Frontend
- **Фреймворк**: React 19
- **Сборка**: Vite 7
- **Язык**: TypeScript

### Mobile
- **Фреймворк**: React Native

### Инфраструктура
- **БД**: PostgreSQL 16
- **Кеш**: Redis 7
- **Контейнеризация**: Docker + Docker Compose
- **Web-сервер**: nginx

##  API Endpoints

### Auth
- `POST /auth/register` — Регистрация
- `POST /auth/login` — Вход по password
- `GET /auth/me` — Получить текущий пользователь
- `GET /auth/qr` — Создать QR-сессию
- `GET /auth/qr/status` — Проверить статус QR
- `POST /auth/qr/confirm` — Подтвердить QR

### Chat
- `GET /chat/messages` — Список сообщений
- `POST /chat/messages` — Отправить сообщение
